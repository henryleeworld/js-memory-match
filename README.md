# JS 記憶配對

先將所有的牌面朝下，可以翻開任何的牌，並找到另一張一樣的牌，二張一樣的牌為一組才算配對成功，如果沒有獲得配對，就把牌蓋回去，要用腦子去記住每張被翻過的卡牌分別是什麼圖案。 

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
